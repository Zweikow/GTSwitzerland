# GT Switzerland

Site web pour les réservations de roadtrips GT Switzerland.

## Description

GT Switzerland organise des roadtrips d'exception à travers l'Europe. Ce site permet aux passionnés d'automobiles de découvrir nos expériences passées et de réserver leur place pour nos prochaines aventures.

## Fonctionnalités

- **Multilingue** : Site disponible en Français, Allemand et Anglais
- **Formulaire de réservation** : Pré-réservation en ligne pour le Roadtrip 2026
- **Formulaire de contact** : Contact direct avec l'équipe
- **Galerie d'expériences** : Présentation des 5 roadtrips réalisés (2021-2025)
- **FAQ** : Réponses aux questions fréquentes
- **Design responsive** : Compatible mobile, tablette et desktop

## Pages

- `accueil.html` - Page d'accueil
- `infos.html` - Actualités et informations
- `experiences.html` - Nos expériences passées
- `voyages.html` - Roadtrip 2026 et réservation
- `contact.html` - Formulaire de contact
- `faq.html` - Questions fréquentes
